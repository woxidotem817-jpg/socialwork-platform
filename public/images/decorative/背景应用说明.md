### 2. 新增组件
**BackgroundWrapper 组件** (`src/components/BackgroundWrapper/BackgroundWrapper.tsx`)
- 支持三种背景类型：`professional`（专业）、`warm`（温暖）、`none`（无）
- 自动调整卡片透明度（95%）以显示背景
- 响应式设计，适配不同屏幕尺寸

## 🎨 视觉效果说明

### 背景容器样式
```
外层容器：
- 背景：应用三维渐变图片
- 尺寸：cover（全覆盖）
- 定位：center（居中）
- 附加：fixed（固定滚动效果）

内层卡片：
- 背景：白色，透明度 95%
- 圆角：12px
- 阴影：0 4px 16px rgba(0, 0, 0, 0.1)
- 最小高度：400px
- 动画：淡入效果（0.3s）
```

### 用户体验优化
1. **内容可读性**：卡片背景 95% 不透明度，确保文字清晰
2. **视觉层次**：通过背景和卡片的对比创造层次感
3. **情感共鸣**：不同页面使用不同背景，传递相应情感
4. **性能优化**：背景图片固定滚动，减少重绘

## 🔧 技术实现

### 使用方法
```tsx
// 导入组件
import BackgroundWrapper from '../components/BackgroundWrapper/BackgroundWrapper'

// 应用背景
return (
  <BackgroundWrapper backgroundType="professional">
    {/* 页面内容 */}
  </BackgroundWrapper>
)
```

### 可选参数
```tsx
<BackgroundWrapper
  backgroundType="professional"  // 'professional' | 'warm' | 'none'
>
  {children}
</BackgroundWrapper>
```
