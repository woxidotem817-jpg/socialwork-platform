## 🚀 快速开始（3步完成）

### 第1步：打开生成器
在浏览器中打开文件：
```
public/images/decorative/generate-images.html
```

直接双击该HTML文件即可在浏览器中打开。

### 第2步：生成图片
1. 页面加载后会自动显示4个装饰文件的预览
2. 点击"**✨ 生成所有图片**"按钮预览效果
3. 如果满意，点击"**💾 下载所有图片**"按钮

### 第3步：移动图片文件
下载的文件会保存在浏览器默认下载文件夹，需要手动移动到：
```
public/images/decorative/
```

确保文件夹中包含以下4个文件：
- ✅ `logo.png` (200×80)
- ✅ `pattern.png` (1024×1024)
- ✅ `gradient-1.jpg` (1920×1080)
- ✅ `gradient-2.jpg` (1920×1080)

## 🎨 生成器功能说明

### 自动生成的内容

| 文件名 | 尺寸 | 设计说明 |
|--------|------|----------|
| logo.png | 200×80 | 心形包裹蛇杖图形 + "安享宁护"文字，体现"专业+人文" |
| pattern.png | 1024×1024 | 浅紫色细线网格 + 小圆点，低存在感增强质感 |
| gradient-1.jpg | 1920×1080 | 浅蓝→淡紫渐变，"专业平静"风格 |
| gradient-2.jpg | 1920×1080 | 暖米色→浅粉渐变，"温暖人文"风格 |

### 设计规范

所有图片严格遵循安宁疗护视觉规范：
- **低饱和度**：所有颜色饱和度≤35%
- **高柔和度**：避免强烈对比，传递温暖、专业氛围
- **简约定制**：无复杂元素、无立体阴影
- **统一风格**：与平台"温暖+专业"核心氛围一致

## 💡 高级选项

### 自定义颜色方案

如果您想调整颜色，可以编辑 `generate-images.html` 文件中的以下参数：

```javascript
// Logo 主色
'#8066D2'  // 淡紫（主色）
'#41A8F9'  // 浅蓝（辅助色）

// Pattern 颜色
'rgba(209, 196, 233, 0.4)'  // 网格颜色

// Gradient 1
'#E8F4F8'  // 浅蓝起点
'#EDE7F6'  // 淡紫终点

// Gradient 2
'#FFE4C4'  // 暖米色起点
'#FCE4EC'  // 浅粉终点
```

修改后刷新浏览器即可看到效果。

## 🔧 故障排除

### 问题1：图片显示为破损图标

**原因**：图片文件未正确下载或移动

**解决**：
1. 重新使用生成器下载图片
2. 确保文件名完全匹配（包括大小写）
3. 检查文件格式是否正确（PNG或JPG）

### 问题2：图片显示模糊

**原因**：图片被拉伸或压缩

**解决**：
1. 检查CSS中是否设置了 `object-fit: contain`
2. 确保图片尺寸符合要求
3. 不要使用CSS强制改变图片尺寸

### 问题3：背景渐变不显示

**原因**：渐变文件未正确应用

**解决**：
1. 确认 `gradient-1.jpg` 和 `gradient-2.jpg` 文件存在
2. 检查CSS路径是否正确
3. 使用浏览器开发者工具检查背景样式

## 📚 扩展使用

### 1. 应用 Pattern 到卡片背景

```tsx
// 在任意卡片组件中
<Card
  style={{
    backgroundImage: 'url(/images/decorative/pattern.png)',
    backgroundRepeat: 'repeat',
  }}
>
  {/* 卡片内容 */}
</Card>
```

### 2. 应用渐变背景到页面

```tsx
// 在页面容器中
<div
  style={{
    backgroundImage: 'url(/images/decorative/gradient-1.jpg)',
    backgroundSize: 'cover',
    backgroundPosition: 'center',
    minHeight: '100vh',
  }}
>
  {/* 页面内容 */}
</div>
```

### 3. 使用 CSS 渐变替代图片（推荐）

```css
/* 专业平静风格 */
.professional-bg {
  background: linear-gradient(135deg, #E8F4F8 0%, #EDE7F6 100%);
}

/* 温暖人文风格 */
.warm-bg {
  background: linear-gradient(135deg, #FFE4C4 0%, #FCE4EC 100%);
}
```
